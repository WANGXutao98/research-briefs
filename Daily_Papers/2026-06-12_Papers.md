## 2. Intercepting the Future: Latent-Space Predictive World Model for Dynamic VLA Manipulation
- **Authors**: (AHEAD, arXiv 2606.02486)
- **ArXiv**: https://arxiv.org/abs/2606.02486
- **PDF**: https://arxiv.org/pdf/2606.02486
- **Published**: 2026-06-02
- **Relevance**: ★★★★★

**Core Innovation**: AHEAD (Anticipatory Horizon Extrapolation with Adaptive Dynamics) adds a 4.9M-parameter motion-aware latent world model to a frozen 7B VLA, forecasting future patch tokens in feature space conditioned on per-token velocity/acceleration, with adaptive rollout horizon halted by prediction uncertainty.

**Why Read**: VLAs fail on moving objects because they assume static scenes between observation and execution — AHEAD solves this with minimal parameter overhead. Results jump from 31–58% (best baseline) to 79–97% across 20 dynamic simulation tasks and achieve 29–30/30 on physical robot tasks where every baseline scores 0/30.

**Research Connection**: Core contribution at the intersection of World Models and VLA; demonstrates that a lightweight predictive world model in feature space is sufficient to make frozen VLAs robust to real-world dynamics without retraining the policy.

## 3. WorldOlympiad: Can Your World Model Survive a Triathlon?

- **PDF**: https://arxiv.org/pdf/2606.11129
- **Published**: 2026-06-09
- **Relevance**: ★★★★☆

**Core Innovation**: A three-track evaluation benchmark for video-based world models: (1) Physical track — does the model obey mechanics, thermal, and material rules (MLLM-as-judge); (2) Geometry track — Gaussian splatting reconstruction to check 3D structural and cross-view consistency; (3) Interactive track — long-horizon controllability under user actions.

**Research Connection**: Directly serves World Model and Video Generation research by providing rigorous evaluation axes (physical plausibility, geometric consistency, long-horizon interactivity) that go beyond FID/FVD metrics toward the kind of grounded assessment needed before deploying world models in Embodied AI systems.

- **Published**: 2026-06-08
- **Relevance**: ★★★★☆

**Core Innovation**: Converts bidirectional diffusion video models into a causal autoregressive paradigm via a unified single-stage training recipe, eliminating the multi-stage pipeline (control fine-tuning → autoregressive training → causal init → distillation) that caused quality degradation and error accumulation in prior interactive world models.

**Research Connection**: Video World Models + Simulation Data — BiWM's single-stage recipe significantly lowers the barrier to building high-quality interactive world models that can serve as data engines or simulators for Embodied AI training.

- **PDF**: https://arxiv.org/pdf/2606.12028
- **Published**: 2026-06-11
- **Relevance**: ★★★★☆

**Core Innovation**: A decoupled closed-loop manipulation framework where a frozen video generation model produces vision-language-conditioned visual plans, and V2T-ICON (Video-to-Trajectory In-Context Operator Network) grounds these plans to robot-state trajectories using retrieved image-state pairs as in-context prompts — no parameter updates at inference.

**Why Read**: VICX cleanly separates semantic planning (video generation) from embodiment-specific execution (operator network), enabling generalization to novel tasks and embodiments without expensive retraining. The in-context grounding mechanism is a practical zero-shot adaptation mechanism.

**Research Connection**: Directly connects Video Generation → Simulation Data → Embodied AI: the video generator acts as a generalizable planner producing synthetic visual trajectories, while the operator network bridges the video-to-action gap — a compelling approach for reducing real-robot data requirements.

## 7. AHA-WAM: Asynchronous Horizon-Adaptive World-Action Modeling with Observation-Guided Context Routing
- **Authors**: Jisong Cai, Long Ling, Shiwei Chu
- **ArXiv**: https://arxiv.org/abs/2606.09811
- **PDF**: https://arxiv.org/pdf/2606.09811
- **Published**: 2026-06-08
- **Relevance**: ★★★★★

**Core Innovation**: Dual Diffusion Transformer (DiT) architecture that decouples world prediction (long-horizon) from action execution (short-horizon) via temporal asymmetry, using observation-guided context routing to selectively inject relevant visual context into each branch.

**Why Read**: Identifies and formalizes a fundamental inefficiency in existing WAMs — coupling world prediction and action execution at the same temporal resolution forces the world branch to model near-term frame variations that are redundant and weakly informative. The asynchronous design is both principled and practically impactful.

**Research Connection**: Core World Action Model architecture contribution; the temporal asymmetry insight has direct implications for how video world models should be structured when used for embodied robot control — relevant to both world model design and VLA post-training.
