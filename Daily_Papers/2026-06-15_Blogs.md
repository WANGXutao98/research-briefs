## Anthropic

### Claude Fable 5 and Claude Mythos 5 *(recent context — June 9, 2026)*
- **Date**: June 9, 2026
- **URL**: https://www.anthropic.com/news/claude-fable-5-mythos-5

**Summary**: Anthropic released Claude Fable 5 as their most capable widely-available model and Claude Mythos 5 as a restricted partner-tier version sharing the same underlying weights. Both models feature a 1 million token context window with up to 128k output tokens per request — a dramatic increase from previous generations. The headline architectural change is *Adaptive Thinking*, which is always-on and non-disableable: the model cannot suppress its chain of thought, though raw thinking content is never returned (only summaries or empty blocks). Fable 5 introduces integrated safety classifiers that return `stop_reason: "refusal"` with a classifier label rather than an error code, enabling structured fallback to other Claude models. Mythos 5 ships without these classifiers and is available only through Project Glasswing to approved biology researchers and enterprise partners. Pricing is $10/M input tokens and $50/M output tokens, and both models carry mandatory 30-day data retention.

**Research Inspiration**: The "adaptive thinking always on" constraint is a strong signal that Anthropic believes persistent, un-bypassable reasoning traces are critical for long-horizon agentic safety — relevant to how embodied agents might need verifiable reasoning chains. The structured refusal taxonomy (classifier label + 200 OK) is a deployable alignment primitive worth studying for world-model agent safety pipelines. The 1M-token context window opens new possibilities for agents that must maintain extended episodic memory across long robot manipulation sequences.

### Paving the Way for Agents in Biology *(recent context — June 8, 2026)*
- **Date**: June 8, 2026
- **URL**: https://www.anthropic.com/research/agents-in-biology

**Summary**: Anthropic published empirical research demonstrating that frontier models (including Claude Sonnet 4 and GPT-5.5) achieve wildly inconsistent accuracy on structured biological data retrieval: 16.9%–91.3% accuracy range on the same viral sequence query tasks across repeated runs. The team built VirBench, a benchmark of 120 real-style virus sequence retrieval tasks across 40 pathogens with manually verified answers drawn from real scenarios (virus surveillance, diagnostic reagent design, protein model training). The core finding is that the bottleneck is not model capability but data infrastructure — existing biological databases are not designed for machine-readable, deterministic agent access. Anthropic collaborated with NCBI to build `gget_virus`, a deterministic retrieval tool that raised agent accuracy to 99.7% and eliminated run-to-run variance entirely.

**Research Inspiration**: This is directly relevant to embodied AI: the lesson is that agent reliability in the real world often depends more on clean, deterministic environment interfaces than on model scale. For world models in physical domains, this suggests that building verifiable, structured environment state representations (rather than querying raw sensor streams) may be the key bottleneck. The VirBench methodology of creating ground-truth benchmarks in specific domains is a transferable evaluation framework for robot manipulation or navigation tasks.

### Gemma 4 12B: Encoder-Free Multimodal AI *(recent context — June 4, 2026)*
- **Date**: June 4, 2026
- **URL**: https://blog.google/innovation-and-ai/technology/developers-tools/introducing-gemma-4-12b/

**Summary**: Google released Gemma 4 12B, described as a unified, encoder-free multimodal model capable of processing text, images, video, and audio without separate vision or audio encoders. At 12B parameters, it is designed to run on 16GB laptops, making it one of the most capable locally-deployable open multimodal models. It is released under Apache 2.0. The encoder-free design is architecturally notable: rather than a separate vision tower (as in LLaVA-style models), visual tokens are processed natively within the decoder.

**Research Inspiration**: Encoder-free multimodal architectures are highly relevant to world models — they suggest a path toward unified scene representations without modality-specific encoders that must be separately trained and aligned. For embodied AI, an encoder-free model that can ingest video, audio, and text in a single pass could form the basis of a more unified world-state estimator. The 12B scale running on a laptop GPU also opens possibilities for on-device robotic inference.

### Gemma 3n: Per-Layer Embeddings for Mobile-Efficient World Models *(recent context — pre-June 13, 2026)*
- **Date**: announced June 2026 (exact date unconfirmed)
- **URL**: https://deepmind.google/models/gemma/gemma-3n/

**Summary**: Gemma 3n introduces Per-Layer Embeddings (PLE), a new architectural technique that dramatically reduces RAM usage by sharing embedding parameters across transformer layers in a non-uniform way. A 5B-parameter model runs with only 2GB active memory footprint; an 8B model runs in 3GB — comparable to much smaller models. The architecture also supports a nested submodel structure, where a 4B model contains a fully capable 2B submodel, enabling dynamic inference-time tradeoffs between quality and speed without loading separate model weights. Gemma 3n handles audio, text, images, and video natively.

**Research Inspiration**: PLE and nested submodel structures are directly applicable to on-device world models for robotics: you could run a fast 2B submodel for reactive control and fall back to the full 4B model for deliberative planning, all within the same weight file. This "mix'n'match" inference is architecturally novel for embedded robotic systems with hard latency/memory constraints. Worth tracking as a deployment pattern for edge embodied AI.

### Cosmos 3: Open Frontier World Model for Physical AI *(recent context — June 1, 2026)*
- **Date**: June 1, 2026
- **URL**: https://nvidianews.nvidia.com/news/nvidia-launches-cosmos-3-the-open-frontier-foundation-model-for-physical-ai
- **Technical report**: https://research.nvidia.com/labs/cosmos-lab/cosmos3/technical-report.pdf

**Summary**: NVIDIA launched Cosmos 3, a fully open world foundation model for physical AI built on a two-tower Mixture-of-Transformers (MoT) architecture that unifies visual reasoning, world simulation, and action generation in a single model. The system exposes two runtime surfaces: *Reasoner* (for understanding, planning, and physical reasoning over images, video, and text) and *Generator* (for producing physics-aware video, ambient sound, and action-conditioned rollouts). Cosmos 3 was trained on 20 trillion multimodal tokens including ~1 billion images, ~400 million real and synthetic videos, ambient audio, and human/robot action data. It is described as the first fully open omnimodel natively unifying vision reasoning and multimodal generation across text, image, video, sound, and action. NVIDIA claims it reduces physical AI training and evaluation cycles from months to days via synthetic data generation.

**Research Inspiration**: This is directly central to David's research focus. Cosmos 3 is the most explicit instantiation of a "world model for embodied AI" from a major lab to date — open weights, physical-reasoning-aware, and natively action-conditioned. The two-tower MoT design separating Reasoner from Generator is architecturally analogous to model-based RL's world model + policy separation, but operationalized at scale. The 400M video training corpus combined with action data suggests that large-scale video pre-training with paired action labels is a viable path to learning physical dynamics. The fact that NVIDIA open-sources this is significant: it is directly usable as a foundation for embodied AI research without needing to train from scratch.
