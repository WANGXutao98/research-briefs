### N-Days: Claude Mythos Preview Turns Disclosed Vulnerabilities Into Working Exploits *(+1 day)*
- **Date**: June 9, 2026
- **URL**: https://red.anthropic.com/2026/n-days/

**Summary**: Anthropic's security research team evaluated Claude Mythos Preview on N-day vulnerability exploitation — publicly disclosed but incompletely patched CVEs. Against 18 Firefox patches, Mythos Preview autonomously built 8 working code-execution exploits (first PoC in ~12 minutes, all 8 in ~12 hours). Against 21 Windows kernel patches (no source code available), it produced PoC triggers for 18 vulnerabilities and 8 full privilege-escalation chains reaching SYSTEM from a low-privilege user, at ~$2,000 per chain. The study concludes that the binding constraint to N-day exploitation is now "a few thousand dollars and API access," dramatically expanding the threat pool.

**Research Inspiration**: The exploit generation pipeline — patch diff → vulnerability hypothesis → exploit synthesis — is architecturally close to a world model that reasons about state transitions in software environments. The same structured reasoning loop (observe delta → infer causal model → test consequence) is central to embodied AI planning under partial observability.

### Introducing Gemma 3n: The Developer Guide
- **Date**: June 10, 2026
- **URL**: https://deepmind.google/blog/introducing-gemma-3n-the-developer-guide/

**Summary**: Google DeepMind released Gemma 3n, a natively multimodal (image, audio, video, text) open model built for on-device inference. The core architectural innovation is MatFormer (Matryoshka Transformer) — a nested transformer that contains smaller fully-functional sub-models, enabling elastic inference. The 5B and 8B parameter models run with memory footprints equivalent to 2B and 4B models (as low as 2GB RAM), making them suitable for phones, tablets, and edge robotics hardware. Supported by Hugging Face, llama.cpp, Ollama, MLX, and Google AI Edge.

**Research Inspiration**: MatFormer's nested elastic inference is highly relevant to embodied AI on resource-constrained platforms (robot onboard compute). The ability to dynamically trade off model size vs. latency without retraining aligns with real-time world model inference on edge devices. The audio-video-text native multimodality is also directly applicable to sensorimotor world models.

- **Date**: June 9, 2026
- **URL**: https://deepmind.google/blog/introducing-the-gemini-25-computer-use-model/

**Summary**: DeepMind released Gemini 2.5 Computer Use, a specialized model built on Gemini 2.5 Pro that enables agents to control user interfaces by interpreting visual screen state. The model reaches ~70% on Online-Mind2Web and strong performance on WebVoyager and AndroidWorld, with lower latency than competing systems (up to 50% latency reduction cited).
