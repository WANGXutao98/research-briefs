## Current Key Research Questions

Based on the accumulated context from yesterday's paper scan, blog scan, research opportunity report, and morning briefing, the live research questions are:

1. **Does visual world model fidelity actually improve policy performance?** AHEAD's 4.9M-parameter result — beating all pixel-level baselines including 7B-scale models on real robot tasks — strongly suggests the field's implicit assumption (higher FVD/FID → better policy) is wrong. This is the core question WorldBench is designed to answer empirically. The question is now: *how wrong is the assumption, and for which task types does it break down most severely?*

2. **What is the right abstraction level for a world model used in VLA?** AHEAD uses frozen VLA backbone patch tokens. Cosmos 3 uses a unified MoE over all modalities. Efficient-WAM treats video as a "guidance signal, not a fidelity target." These three approaches represent three distinct answers to the same question. Which wins on downstream task success per parameter? This is the "Latent Abstraction Scaling Laws" question from yesterday's opportunity report.

3. **What is the minimal viable world model extension for `sana-wm-pipeline`?** The existing pipeline focuses on simulation data annotation. After reading AHEAD, the specific question becomes: can the latent world model architecture (motion-aware feature prediction + uncertainty-halted horizon) be grafted onto the existing annotation pipeline as a data quality filter — keeping only trajectories where the world model's uncertainty is low (confident transition), which may correlate with data annotation quality?

4. **Can the WorldBench WMATS metric be defined precisely enough to be peer-reviewable?** "World Model Action Transfer Score" is currently a conceptual label. The gap between the concept and a concrete, reproducible metric definition is the key intellectual work this week.
